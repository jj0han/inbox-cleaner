# Plan 07-02: SuggestionsSection Error State

## Goal

When `getSuggestions` query fails, replace the `SuggestionsSection` body with an inline error card containing an icon, a descriptive message, and a "Tentar novamente" button that calls `refetch`. Currently the component swallows errors silently (returns `[]` on failure from the server, so the UI just shows empty state — no recovery option).

---
