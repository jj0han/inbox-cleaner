## v1 Requirements — Final Status

### Authentication
- [x] **AUTH-01**: User can securely connect their Gmail account via OAuth — *Phase 1: NextAuth + Google provider*
- [x] **AUTH-02**: System synchronizes inbox metadata without downloading full message bodies — *Phase 1: Only headers + metadata fetched*

### Dashboard
- [x] **DASH-01**: User can view a Hero section showing total emails available to clean immediately — *Phase 2 + Phase 4 fix: Now shows accurate INBOX count via `messages.list` with INBOX label*
- [x] **DASH-02**: User can view Action Cards summarizing potential cleanups — *Phase 2: Cards with phase 4 live count badges*

### Automations
- [x] **AUTO-01**: User can execute a "Mass Unsubscribe" action using detected patterns/links — *Phase 3: `has:list-unsubscribe` query + batchModify*
- [x] **AUTO-02**: User can execute an "Archive Promos" action — *Phase 3: `category:promotions` query*
- [x] **AUTO-03**: User can use the "Intelligent 1-Click Cleanup" button — *Phase 3: Combined promotions+social+updates older_than:30d*
- [x] **AUTO-04**: User can set up suggested auto-archive rules — *Phase 4: getSuggestions + applySuggestion + Gmail filter creation*

### Trust & Safety
- [x] **SAFE-01**: User can view a Smart Preview of 5-10 emails before executing any bulk action — *Phase 2: PreviewModal*
- [x] **SAFE-02**: User can click "Undo" within 10-30 seconds after any bulk action to revert it — *Phase 3: 30s countdown toast + batchModify restore; Phase 4: extends undo to delete Gmail filter*
